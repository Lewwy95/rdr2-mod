## About

rdr2-mod is a custom mod manager for Red Dead Redemption 2.

## Requirements

* Red Dead Redemption 2 (Steam)

## Installation

Place the 'rdr2-mod' folder in your Red Dead Redemption 2 game installation directory then run the following file:

```
install.bat
```
